# File Organizer

A program that organizes your files into folders according to their respective types.

---

## Features

*   **Auto Find Path**
    *   Auto searches for your downloads path.
*   **Works with any Path**
    *   You can manually type any path of your choice for the program to work.
*   **Auto Create Folders**
    *   The program auto create folders or check if they already exists so it doesn't make duplicates.
*   **Extension Sorting**
    *   It also auto sorts files by its extension (e.g., .png -> images folder).
*   **Robust Code**
    *   The program deals with input\directories validation ensuring no crashes.

---

*   **Language:** Java

## How to Run

1.  **Prerequisites:** You need to have the Java Development Kit (JDK) installed on your machine.
2.  **Clone the repository:**
    ```bash
    git clone https://github.com/Tzavi727/File-Organizer.git
    ```
3.  **Navigate to the source folder and compile:**
    ```bash
    javac com/tzavi/fileorganizer/FileOrganizer.java
    ```
4.  **Run the application:**
    ```bash
    java com.tzavi.fileorganizer.FileOrganizer
    ```
*(Note: Running directly from an IDE like VS Code or IntelliJ is the recommended way.)*
